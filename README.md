# harbor-tasks

Synthetic image assets for Harbor multimodal task contributions.

## Tasks

### `spot-the-difference`
Synthetic top-down warehouse floor scenes (before/after).
Generated by `generate_images.py` using Pillow and NumPy.

### `reconstruct-the-chart`
Synthetic grouped bar chart image.
Generated by `generate_chart.py` using matplotlib and NumPy.
