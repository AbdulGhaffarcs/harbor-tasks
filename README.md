# harbor-tasks

Synthetic image assets for Harbor multimodal task contributions.

## Contents

### `generate_images.py`
Procedurally generates `before.png` and `after.png` — synthetic top-down
warehouse floor scenes used in the **Spot the Difference** task.

- No third-party image assets used
- Built with Pillow and NumPy only
- Output images released under CC0
